Property:
1) (a+b) % c = (a%c + b%c) % c
2) (a-b) % M = (a%c - b%c) % c     (for Negative cycle)
3) (a-b) % M = (a%c - b%c +c) % c  (for Positive cycle)
4) (a*b) % M = (a%c * b%c) % c
5) (a/b) % M Not applicable
